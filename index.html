Instructions

Some important instructions you
need to follow to play the quiz

Click this if you already know the instructions

There will be 5 levels with 10 questions in each level.
Time limit to complete each level will be indicated in the top right corner.
You’ll get 100 JJ coins for each correct answer. (100 JJ coins = 1 Re)
To unlock the next level, make sure you score 60% or more.
Questions will be displayed one at a time. Each question has several options, but only one is correct.
Make sure to complete all the levels within 24 hours of the start of the quiz, or you’ll have to start from scratch.
That’s all!! Good Luck!
